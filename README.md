# github-final-project
This project calculates simple interest using the formula:

Simple Interest = (P × R × T) / 100

## Inputs
- P = Principal amount
- R = Rate of interest (per year)
- T = Time (in years)

## Example
If P = 1000, R = 5%, T = 2 years:

SI = (1000 × 5 × 2) / 100 = 100

## How to use
Enter values of P, R, and T to calculate simple interest.
